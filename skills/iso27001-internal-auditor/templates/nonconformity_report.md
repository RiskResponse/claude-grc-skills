## INSTRUCTIONS FOR USE:

1. Create one NCR for each nonconformity
2. Complete all sections - especially evidence and root cause
3. Assign specific responsible party (name and role)
4. Set realistic deadlines based on severity
5. Create corresponding Vanta task
6. Track progress with status updates
7. Verify effectiveness before closing
8. Obtain closure approval from ISMS manager
9. Keep for ISMS records (minimum 3 years)
10. Review during next internal audit
