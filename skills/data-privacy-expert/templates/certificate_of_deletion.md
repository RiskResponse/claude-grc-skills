# USAGE INSTRUCTIONS

1. **Make a copy** of this template for each deletion request
2. **Rename file** to include certificate number and date: `Certificate_[CERT-ID]_[YYYY-MM-DD].md`
3. **Complete all sections** - do not leave fields blank; use "N/A" if not applicable
4. **Check all applicable boxes** using [X]
5. **Remove internal instructions** before finalizing (all sections in gray or marked as instructions)
6. **Do NOT include actual personal data** in the certificate itself - use categories and descriptions
7. **Obtain required signatures** before issuance
8. **Deliver to data subject** within regulatory timelines
9. **File securely** with other compliance documentation
10. **Set retention schedule** per applicable laws
